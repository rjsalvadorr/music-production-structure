# Original Beats

Contains homemade sound files and MIDI files.

## MIDI Mappings

The MIDI files use these mappings, derived from an AKAI MPK mini controller: 

```
+------------+--------------+-------------+-------------+
| [48] SNARE | [49] SNARE   | [50] HI-HAT | [51] HI-HAT |
|            |      (GHOST) |      CLOSED |      OPEN   |
+------------+--------------+-------------+-------------+
| [44] KICK  | [45] SNARE   | [46] ???    | [47] RIDE   |
|            |      (STICK) |             |             |
+------------+--------------+-------------+-------------+
```
